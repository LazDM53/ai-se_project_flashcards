# Flashcard App

This is a simple and interactive Flashcard App that i made with the TripleTen lessons. It is made to help students study and memorize information. The app allows you to browse decks, flip the cards in the decks to reveal answers, and navigate through questions with a carousel interface.

I built this app to practice front-end development skills like DOM manipulation and routing. It solves the problem of needing a lightweight, distraction-free study tool that runs directly in the browser without requiring any installation.

## Features

- 📂 View multiple flashcard decks
- 🔄 Flip cards to reveal answers
- ⬅️➡️ Navigate between cards using a carousel
- 🧭 Hash-based routing (no page reloads)
- 💾 Persistent data using LocalStorage (in progress / planned)
- 🎨 Dynamic color theming using a color map

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, BEM methodology)
- JavaScript (ES6 modules)
- LocalStorage (planned/partial)
- Git & GitHub for version control
